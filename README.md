## Devcontainer for LaTeX Projects in Coder

### Help needed!

This is my first devcontainer for Coder https://coder.com. Most of the time a wrote in LaTeX. I prefer to use TeX Live.

The GitHub-Repo for this devcontainer can you find here: https://github.com/joergklein/devcontainer-latex

1. Use a template "Docker (Devcontainer)"
1. Please clone this repo and install it in Coder (docker-compose up -D)
1. Please take a look at the Dockerfile and docker-compose.yaml. Are they correct?
1. Use code-server and visual studio Code
1. Compile some of your LaTeX code. Tables, pictures etc.
1. Send my your experience with this devcontainer

Thank you for the help.
